Core idea: 4 TextBoxes + 1 Button. On click, parse all 4 as int, multiply, show result with Response.Write.
1
Open Visual Studio → New Project
File → New → Project → ASP.NET Web Application (.NET Framework) → Name it → Select "Web Forms" template → OK.
Use WebForms not MVC. Make sure .NET Framework is selected, not .NET Core.

Open About.aspx

PRACTICAL 2
Phone Book with SQL
INSERT + SELECT with SqlConnection — the core pattern for the exam
MEDIUM
Step-by-Step
Full Code
Memory Cards
This is the most important pattern to know. SqlConnection → SqlCommand → Parameters → Open → Execute → Close. Memorize this sequence.
1
Create the Database in SQL Server
Open SQL Server Management Studio (SSMS). In the Connect dialog, type (localdb)\MSSQLLocalDB as the server name → click Connect. In Object Explorer (left panel), right-click Databases → New Database → name it mydb → OK. Then open a New Query and run:
CREATE DATABASE mydb; USE mydb; CREATE TABLE contacts ( id INT PRIMARY KEY IDENTITY, name NVARCHAR(100), phonenumber NVARCHAR(15) );
🖱️ SSMS UI tip: After running the query, right-click Tables under mydb in Object Explorer → Refresh to see your new table appear. You can right-click the table → Select Top 1000 Rows to visually verify data after running the app.
2
Set the Connection String
In your code-behind, define the connection string as a field. This tells C# where your database lives.
string connStr = @"Data Source=(localdb)\MSSQLLocalDB; Initial Catalog=mydb; Integrated Security=True;";
🖱️ VS tip: You can also find the exact connection string by going to View → Server Explorer → right-click Data Connections → Add Connection → browse to your DB → after connecting, click the connection and check the Properties panel (F4) — it shows the connection string ready to copy.
3
Design the ASPX Page
Open Default.aspx. In Design view, drag from Toolbox: a TextBox (for name), another TextBox (for phone), a RegularExpressionValidator, a Button, a Label, and a GridView.
🖱️ GridView smart tag: After dragging a GridView onto the designer, a small ▶ arrow (smart tag) appears at its top-right corner. Click it → you'll see options like "Auto Format", "Choose Data Source", "Edit Columns". Use Edit Columns to add BoundField columns visually instead of typing them in Source view.
4
Write the INSERT Logic (BtnSubmit_Click)
Double-click the Button in Design view to auto-generate the click handler in code-behind. Use parameterized queries — never concatenate strings directly (SQL injection risk).
SqlConnection con = new SqlConnection(connStr); string str = "INSERT INTO contacts (name, phonenumber) VALUES (@name, @phonenumber)"; SqlCommand cmd = new SqlCommand(str, con); cmd.Parameters.AddWithValue("@name", name.Text); cmd.Parameters.AddWithValue("@phonenumber", phonenum.Text); con.Open(); cmd.ExecuteNonQuery(); con.Close();
5
Write the SELECT Logic (LoadContacts)
Fetch all records and bind to GridView. Use ExecuteReader for SELECT queries.
private void LoadContacts() { SqlConnection con = new SqlConnection(connStr); SqlCommand cmd = new SqlCommand("SELECT name, phonenumber FROM contacts", con); con.Open(); GridViewContacts.DataSource = cmd.ExecuteReader(); GridViewContacts.DataBind(); con.Close(); }
6
Call LoadContacts on Page_Load
Use IsPostBack to avoid reloading on every button click postback.
protected void Page_Load(object sender, EventArgs e) { if (!IsPostBack) LoadContacts(); }
🖱️ Shortcut: In Design view, double-click an empty area of the page (not a control) to auto-generate the Page_Load handler in code-behind.

Default.aspx.cs
using System;
using System.Data.SqlClient;
using System.Web.UI;

namespace WebApplication
{
    public partial class _Default : Page
    {
        string connStr = @"Data Source=(localdb)\MSSQLLocalDB;Initial Catalog=mydb;Integrated Security=True;";

        protected void Page_Load(object sender, EventArgs e) 
        { 
            if (!IsPostBack) LoadContacts(); 
        }

        protected void BtnSubmit_Click(object sender, EventArgs e)
        {
            SqlConnection con = new SqlConnection(connStr);
            string str = "INSERT INTO contacts (name, phonenumber) VALUES (@name, @phonenumber)";
            SqlCommand cmd = new SqlCommand(str, con);
            cmd.Parameters.AddWithValue("@name", name.Text);
            cmd.Parameters.AddWithValue("@phonenumber", phonenum.Text);
            con.Open();
            cmd.ExecuteNonQuery();
            con.Close();
            LoadContacts();
        }

        private void LoadContacts()
        {
            SqlConnection con = new SqlConnection(connStr);
            SqlCommand cmd = new SqlCommand("SELECT name, phonenumber FROM contacts", con);
            con.Open();
            GridViewContacts.DataSource = cmd.ExecuteReader();
            GridViewContacts.DataBind();
            con.Close();
        }
    }
}

Q2 · Core Pattern
What are the 5 steps of the SQL pattern in order?
1. new SqlConnection(connStr)
2. new SqlCommand(query, con)
3. cmd.Parameters.AddWithValue()
4. con.Open()
5. cmd.ExecuteNonQuery() then con.Close()
Q2 · Key Difference
ExecuteNonQuery vs ExecuteReader — when to use each?
ExecuteNonQuery() → INSERT, UPDATE, DELETE (no data returned)
ExecuteReader() → SELECT (returns rows of data)
Q2 · GridView
How do you display SQL data in a GridView?
GridView.DataSource = cmd.ExecuteReader();
GridView.DataBind();
The DataField in BoundField must match the column name in SQL.
Q2 · PostBack
Why check "if (!IsPostBack)" in Page_Load?
Page_Load fires on every request including button clicks. Without this check, you'd reload data even when you don't need to, causing issues.
PRACTICAL 2B
Pythagoras Theorem
Enter 2 sides → calculate missing side using Math.Sqrt
EASY
Step-by-Step
Code
Core idea: a² + b² = c² — if the user fills 2 fields, calculate the 3rd. Use Math.Sqrt() and double.TryParse() to safely read inputs.
1
Create 3 TextBoxes
One for hypotenuse (c), one for side a, one for side b. Leave one empty — that's the one to calculate.
🖱️ Design view tip: Open Toolbox → Standard → drag 3 TextBox controls onto the page. In the Properties panel (F4), set the ID of each: hypotenuse, sidea, sideb. Also drag a Button and a Label (ID: lblResult).
2
Use double.TryParse to read values
TryParse sets the variable to 0 if the field is empty — perfect for detecting which side is missing.
double a = 0, b = 0, c = 0; double.TryParse(hypotenuse.Text, out a); double.TryParse(sidea.Text, out b); double.TryParse(sideb.Text, out c);
3
Write if-else for each missing side
If a (hypotenuse) is 0, calculate it. If c (side b) is 0, calculate it.
if (b > 0 && c > 0 && a == 0) { a = Math.Sqrt((b * b) + (c * c)); lblResult.Text = "Hypotenuse = " + a.ToString("0.00"); } else if (a > 0 && b > 0 && c == 0) { c = Math.Sqrt((a * a) - (b * b)); lblResult.Text = "Side B = " + c.ToString("0.00"); }
Note: a.ToString("0.00") formats to 2 decimal places.
BtnSubmit_Click logic
protected void BtnSubmit_Click(object sender, EventArgs e)
{
    double a = 0, b = 0, c = 0;
    double.TryParse(hypotenuse.Text, out a);
    double.TryParse(sidea.Text, out b);
    double.TryParse(sideb.Text, out c);

    if (b > 0 && c > 0 && a == 0) {
        a = Math.Sqrt((b * b) + (c * c));
        lblResult.Text = "Hypotenuse = " + a.ToString("0.00");
    } else if (a > 0 && b > 0 && c == 0) {
        c = Math.Sqrt((a * a) - (b * b));
        lblResult.Text = "Side B = " + c.ToString("0.00");
    }
}

Fibonacci, Prime, Vowel, Reverse, Digit Sum
Logic snippets — know the pattern for each

Logic Snippets
Memory Cards
These are all code-behind only. The ASPX is just TextBoxes + a Button + a Label. All logic lives in the button click handler.
🖱️ VS UI tip for all Q3 sub-practicals: In Design view, drag one TextBox (ID: txtInput), one Button, and one Label (ID: lblResult) from the Toolbox onto the page. Double-click the Button to auto-generate the click handler. Then write the logic below inside that handler. Use the Properties panel (F4) to set IDs without touching Source view.
Fibonacci Series
int a = 0, b = 1, c = 0;
int limit = int.Parse(txtInput.Text);
string series = a + " " + b;
for (int i = 2; i < limit; i++) {
    c = a + b;
    series += " " + c;
    a = b; b = c;       // shift: a becomes b, b becomes new c
}
lblResult.Text = series;
Prime Number Test
int num = int.Parse(txtInput.Text);
int f = 0;
for (int i = 1; i <= num; i++) {
    if (num % i == 0) f++;   // count factors
}
if (f == 2)                   // prime = exactly 2 factors (1 and itself)
    lblResult.Text = "Prime";
else
    lblResult.Text = "Not Prime";
Vowel Test
char ch = char.ToLower(txtInput.Text[0]);
switch (ch) {
    case 'a': case 'e': case 'i': case 'o': case 'u':
        lblResult.Text = "Vowel"; break;
    default:
        lblResult.Text = "Consonant"; break;
}
Reverse Number + Sum of Digits
int num = int.Parse(txtInput.Text);
int rev = 0, sum = 0, temp = num;
while (temp > 0) {
    int digit = temp % 10;   // get last digit
    rev = rev * 10 + digit;  // build reverse
    sum += digit;             // accumulate sum
    temp /= 10;              // remove last digit
}
lblResult.Text = "Reversed: " + rev + ", Sum: " + sum;
Q3 · Fibonacci
What are the starting values for Fibonacci?
a = 0, b = 1 — series starts "0 1 1 2 3 5 8...". Each new number c = a + b, then shift: a = b, b = c.
Q3 · Prime
How do you check if a number is prime?
Count its factors. If num % i == 0, it's a factor. A prime has exactly 2 factors (1 and itself), so if (f == 2) → Prime.
Q3 · Reverse
How do you reverse a number using a while loop?
digit = temp % 10 (last digit)
rev = rev * 10 + digit (build reverse)
temp /= 10 (remove last digit)
Repeat until temp == 0.
Q3 · Vowel
What C# method gets the first character of a string?
txtInput.Text[0] — index 0. Use char.ToLower() to make it case-insensitive before the switch statement.

int factinput;
int.TryParse(factorial.Text, out factinput);
int f = 1;
for (int i = 1; i <= factinput; i++) f = f * i;
lblResult.Text = "Factorial of " + factinput + "! is: " + f;

// CURRENCY
double amount;
double.TryParse(money.Text, out amount);
if (currency.Text == "jpy")
    lblResult1.Text = "Result: inr " + (0.57 * amount);
else
    lblResult1.Text = "Result: jpy " + (1.76 * amount);

// QUADRATIC
int num1, num2, num3;
int.TryParse(number1.Text, out num1);
int.TryParse(number2.Text, out num2);
int.TryParse(number3.Text, out num3);
if (num1 == 0) { lblResult2.Text = "Invalid"; }
int sqr = num2 * num2 - 4 * num1 * num3;
double sqrt_val = Math.Abs(sqr);
if (sqr > 0)
    lblResult2.Text = "Roots are real: " +
        (-num2 + sqrt_val)/(2*num1) + " and " + (-num2 - sqrt_val)/(2*num1);
else
    lblResult2.Text = "Roots are complex: " +
        (-(double)num2/(2*num1)) + " ± i" + sqrt_val/(2*num1);

// TEMPERATURE
double numbergiven;
double.TryParse(temp.Text, out numbergiven);
if (fc.Text == "F")
    lblResult3.Text = "Celsius: " + ((numbergiven - 32) * 5.0 / 9.0) + "C";
else
    lblResult3.Text = "Fahrenheit: " + (numbergiven * 9 / 5 + 32) + "F";
Q3b · Factorial
What's the starting value of f in factorial?
int f = 1 — NOT 0! (0 × anything = 0). Loop: for(i=1; i<=input; i++) f = f * i;
Q3b · Quadratic
What does Math.Abs() do in the quadratic calculation?
Gets the absolute (positive) value of the discriminant (b²-4ac) before taking its square root — prevents errors with negative values when computing complex roots.
Q3b · Currency
How do you check which currency to convert?
if(currency.Text == "jpy") → convert to INR (×0.57). Else convert to JPY (×1.76). Plain string comparison on the TextBox value.
Q3b · Temperature
Celsius to Fahrenheit formula?
F = C × 9/5 + 32. Fahrenheit to Celsius: C = (F-32) × 5/9. Check fc.Text == "F" to know input is Fahrenheit.

Q4 · Validators
What's the difference between RequiredFieldValidator and RangeValidator?
Required = field cannot be empty. Range = numeric value must fall between min and max. Use both on the Age field.
Q4 · DropDown
How do you validate a DropDownList?
Add RequiredFieldValidator with InitialValue="" — this matches the placeholder item's Value. Triggers if user never changes from --Select--.
Q4 · Page.IsValid
Why check Page.IsValid in code-behind?
Even after client-side validation, always check server-side: if (Page.IsValid). Prevents invalid data if JavaScript is disabled.
Q4 · RadioButtonList
How do you read the selected radio button value?
rblGender.SelectedValue — returns the text of the chosen ListItem (e.g. "Male" or "Female").

protected void GridViewStudents_RowUpdating(object sender, GridViewUpdateEventArgs e)
{
    int id = Convert.ToInt32(GridViewStudents.DataKeys[e.RowIndex].Value);
    string name = ((TextBox)GridViewStudents.Rows[e.RowIndex].Cells[1].Controls[0]).Text;
    
    SqlConnection con = new SqlConnection(connStr);
    string query = "UPDATE students SET name=@n WHERE id=@id";
    SqlCommand cmd = new SqlCommand(query, con);
    cmd.Parameters.AddWithValue("@n", name);
    cmd.Parameters.AddWithValue("@id", id);
    con.Open();
    cmd.ExecuteNonQuery();
    con.Close();
    
    GridViewStudents.EditIndex = -1;
    LoadStudents();
}

Default.aspx.cs — Disconnected Pattern
private void LoadStudents()
{
    using (SqlConnection con = new SqlConnection(connStr))
    {
        SqlDataAdapter da = new SqlDataAdapter("SELECT * FROM students", con);
        SqlCommandBuilder cb = new SqlCommandBuilder(da);
        DataSet ds = new DataSet();
        da.Fill(ds, "students");
        ViewState["DataSet"] = ds;         // store offline
        GridViewStudents.DataSource = ds.Tables["students"];
        GridViewStudents.DataBind();
    }
}

protected void BtnSubmit_Click(object sender, EventArgs e)
{
    DataSet ds = (DataSet)ViewState["DataSet"];
    DataRow row = ds.Tables["students"].NewRow();
    row["name"] = txtName.Text;
    row["age"]  = txtAge.Text;
    row["department"] = ddlDept.SelectedItem.Text;
    row["gender"]     = rblGender.SelectedItem.Text;
    ds.Tables["students"].Rows.Add(row);
    using (SqlConnection con = new SqlConnection(connStr))
    {
        SqlDataAdapter da = new SqlDataAdapter("SELECT * FROM students", con);
        SqlCommandBuilder cb = new SqlCommandBuilder(da);
        da.Update(ds, "students");         // push to DB
    }
    ViewState["DataSet"] = ds;
    LoadStudents();
}

protected void GridViewStudents_RowUpdating(object sender, GridViewUpdateEventArgs e)
{
    DataSet ds = (DataSet)ViewState["DataSet"];
    int index = e.RowIndex;
    DataRow row = ds.Tables["students"].Rows[index];
    row["name"] = ((TextBox)GridViewStudents.Rows[index].FindControl("txtEditName")).Text;
    row["age"]  = ((TextBox)GridViewStudents.Rows[index].FindControl("txtEditAge")).Text;
    row["department"] = ((TextBox)GridViewStudents.Rows[index].FindControl("txtEditDept")).Text;
    row["gender"]     = ((TextBox)GridViewStudents.Rows[index].FindControl("txtEditGender")).Text;
    using (SqlConnection con = new SqlConnection(connStr))
    {
        SqlDataAdapter da = new SqlDataAdapter("SELECT * FROM students", con);
        new SqlCommandBuilder(da);
        da.Update(ds, "students");
    }
    ViewState["DataSet"] = ds;
    GridViewStudents.EditIndex = -1;
    LoadStudents();
}

protected void GridViewStudents_RowDeleting(object sender, GridViewDeleteEventArgs e)
{
    DataSet ds = (DataSet)ViewState["DataSet"];
    ds.Tables["students"].Rows[e.RowIndex].Delete();
    using (SqlConnection con = new SqlConnection(connStr))
    {
        SqlDataAdapter da = new SqlDataAdapter("SELECT * FROM students", con);
        new SqlCommandBuilder(da);
        da.Update(ds, "students");
    }
    ViewState["DataSet"] = ds;
    LoadStudents();
}
Q5b · Disconnected
What does "disconnected" mean in ADO.NET?
Data is loaded into a DataSet in memory — the DB connection is closed. Changes are made offline, then pushed back with da.Update().
Q5b · DataSet
How is DataSet different from DataReader?
DataReader = forward-only, connection stays open, one row at a time. DataSet = full table in memory, connection closed, random access, can be stored in ViewState.
Q5b · SqlCommandBuilder
What does SqlCommandBuilder do?
Auto-generates INSERT/UPDATE/DELETE SQL commands for the DataAdapter — you don't write them manually. Required when using da.Update().
Q5b · ViewState
Why store DataSet in ViewState?
ASP.NET is stateless — each postback loses memory. ViewState persists data between postbacks without re-querying the DB.

MongoHelper.cs (full)
using MongoDB.Bson;
using MongoDB.Bson.Serialization.Attributes;
using MongoDB.Driver;
using System.Collections.Generic;
using System.Linq;

public class MongoHelper
{
    private readonly IMongoDatabase db;
    private readonly IMongoCollection<User> users;

    public MongoHelper()
    {
        var client = new MongoClient("mongodb://localhost:27017");
        db = client.GetDatabase("MyDatabase");
        users = db.GetCollection<User>("Users");
    }

    public void InsertUser(User user) => users.InsertOne(user);

    public List<User> GetAllUsers() => 
        users.Find(FilterDefinition<User>.Empty).ToList();

    public void UpdateUser(string id, User updatedUser)
    {
        var filter = Builders<User>.Filter.Eq(u => u.Id, id);
        users.ReplaceOne(filter, updatedUser);
    }

    public void DeleteUser(string id)
    {
        var filter = Builders<User>.Filter.Eq(u => u.Id, id);
        users.DeleteOne(filter);
    }
}

public class User
{
    [BsonId]
    [BsonRepresentation(BsonType.ObjectId)]
    public string Id { get; set; }
    public string Name { get; set; }
    public int Age { get; set; }
}
Q7 · MongoDB
What does [BsonId] do?
Marks the property as the MongoDB document's _id field. [BsonRepresentation(BsonType.ObjectId)] lets you store the ObjectId as a string in C#.
Q7 · MongoDB
How do you create a filter for a specific document?
Builders<User>.Filter.Eq(u => u.Id, id) — Eq means "equals". This finds documents where the Id field matches.
Q7 · MongoDB
How do you get ALL documents from a collection?
users.Find(FilterDefinition<User>.Empty).ToList() — Empty filter means no condition = return everything.
Q7 · Structure
What is the purpose of the MongoHelper class?
It's a wrapper/helper that handles the MongoDB connection and all CRUD methods. The code-behind just calls its methods, keeping DB logic separate.
